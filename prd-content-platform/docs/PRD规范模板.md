# 交互题 PRD 规范模板 v1.0

> 本规范定义了交互题 PRD 的标准格式。AI 平台将按此格式解析并自动生成全部素材。
> 未填写的字段视为"无需求"，可省略。

---

## 一、产品信息（必填）

```
产品线：探索自然 / 百科5+ / 揭秘疾病 / ...
集数：E1
主题名称：动物社群
背景风格：自然纪录片实拍风格 / 卡通2D插画 / ...
```

## 三、各字段详细说明

### 3.1 基本信息

| 字段 | 格式 | 必填 | 说明 |
|------|------|------|------|
| 题号 | E1-1, E2-3 等 | 是 | Epic编号-题序号 |
| 题型 | 见下方枚举 | 是 | 决定布局模板和交互逻辑 |
| 正确答案 | A / A、B、C / B→A→C | 是 | 单选写单个，多选逗号分隔，拖拽用箭头标顺序 |
| 考查点 | 自由文本 | 否 | 用于元数据记录 |

**支持的题型：**
- `单选题` — 选择一个正确答案
- `多选题` — 选择多个正确答案
- `拖拽题` — 将选项拖到目标区域（支持排序）
- `连线题` — 将左右两侧元素配对连接
- `点选题` — 在图片上点击特定区域（热区/不规则点选）

### 3.2 题干

| 字段 | 格式 | 说明 |
|------|------|------|
| 题干 | 文字内容 | 题目的问题描述文字 |
| 题干图 | `媒体类型 \| 图片描述` 或 `无` | 题干区域的配图 |

**题干图的媒体类型：**
- `实拍图` — AI 生成自然纪录片风格的真实照片
- `2d插画` — AI 生成儿童教育风格的卡通插画
- `无` — 没有题干图

**格式示例：**
```
- 题干：织叶蚁形成活锁链时，单只蚁的拉力会有什么变化呢？
- 题干图：实拍图 | 织叶蚁形成活锁链的特写照片，多只蚂蚁首尾相连悬挂在两片树叶之间
```

### 3.3 选项

每个选项一行，格式：

```
选项字母：选项文字 | 媒体类型 | 图片描述（仅有图时填写）
```

**媒体类型：**
- `文字` — 纯文字选项，无图片
- `文字+拼音` — 文字附拼音标注（低龄产品线）
- `实拍图+文字` — 自然照片 + 文字标签
- `2d插画+文字` — 卡通插画 + 文字标签
- `实拍图` — 仅图片，无文字
- `2d插画` — 仅插画，无文字
- `实拍图+语音` — 图片 + 点击播放语音
- `2d插画+语音` — 插画 + 点击播放语音
- `文字+语音` — 文字 + 点击播放语音

**格式示例：**
```
- 选项：
  - A：合力拉树叶 | 实拍图+文字 | 多只织叶蚁合力拉动一片绿色树叶，特写镜头
  - B：找合适位置 | 实拍图+文字 | 一只织叶蚁在树枝分叉处观察，寻找合适筑巢位置
  - C：粘合树叶 | 实拍图+文字 | 织叶蚁口衔幼虫，用幼虫吐出的丝将两片叶缘粘合在一起
```

### 3.4 正确解析

```
- 正确解析：回答正确后显示的解析文字内容
```

### 3.5 动效需求

#### 3.5.1 开场动效（进入题目时播放）

```
- 开场动效：
  - 作用对象：题干图 / 背景 / 全屏
  - 效果描述：[详细描述画面运动内容，作为AI视频生成的提示词]
  - 时长：4秒 / 6秒 / 8秒
```

**如无开场动效写：**
```
- 开场动效：无
```

#### 3.5.2 正确反馈动效（回答正确后播放）

```
- 正确反馈动效：
  - 作用对象：题干图 / 选项A / 选项B / 背景 / 全屏
  - 效果描述：[详细描述画面运动内容]
  - 时长：4秒 / 6秒 / 8秒
```

**作用对象说明：**
| 作用对象 | 含义 | 对应 config.json 字段 |
|----------|------|----------------------|
| 题干图 | 题干区域的图片产生动画 | endAnimations / endBackgroundPictures |
| 选项A/B/C | 某个选项图片产生动画 | endAnimations |
| 背景 | 整个背景替换或产生动画 | endBackgroundPictures |
| 全屏 | 全屏播放动画 | endAnimations |

**效果描述写作要求（直接用于 AI 视频生成）：**
- 主体：描述动画的主角是什么（如"雄狮"）
- 动作：描述具体动作（如"张开嘴巴露出牙齿，发出吼叫"）
- 镜头：描述镜头运动（如"特写镜头缓慢推近"）
- 环境：描述背景环境（如"非洲草原，黄昏光线"）
- 风格：如果与整体风格不同，单独注明

**格式示例：**
```
- 正确反馈动效：
  - 作用对象：题干图
  - 效果描述：织叶蚁活锁链从静止开始轻微晃动，蚂蚁们抓紧彼此的腿部，镜头从远景缓推至蚂蚁抓握细节的特写，背景为模糊的绿色树叶
  - 时长：4秒
```

```
- 正确反馈动效：
  - 作用对象：选项A
  - 效果描述：选项A的雄狮图片放大至全屏，雄狮张开嘴巴露出牙齿发出吼叫，鬃毛随风飘动，背景为非洲草原黄昏
  - 时长：4秒
```

#### 3.5.3 错误反馈动效（回答错误后播放）

```
- 错误反馈动效：
  - 作用对象：[同上]
  - 效果描述：[同上]
  - 时长：4秒 / 6秒 / 8秒
```

**如无错误反馈动效写：**
```
- 错误反馈动效：无
```

### 3.6 配音需求

#### 3.6.1 题干配音

```
- 题干配音：[朗读的文字内容] | [播放时机]
```

**播放时机：**
- `进入时自动播放` — 进入题目后自动朗读
- `点击播放按钮` — 用户点击题干区播放按钮时朗读
- `无` — 不需要题干配音

#### 3.6.2 选项配音

```
- 选项配音：
  - A：[朗读文字] | [播放时机]
  - B：[朗读文字] | [播放时机]
  - C：[朗读文字] | [播放时机]
```

**播放时机：**
- `点击时播放` — 点击该选项时朗读
- `自动播放` — 选项出现时依次朗读
- `无` — 不需要配音（纯文字选项通常不需要）

**如所有选项都无配音写：**
```
- 选项配音：无
```

#### 3.6.3 正确解析配音

```
- 正确解析配音：[朗读文字] | 答对后播放
```

#### 3.6.4 错误提示配音

```
- 错误提示配音：[朗读文字] | 答错后播放
```

#### 3.6.5 语气要求（每题或全局统一设置）

```
- 语气要求：儿童科普风格，语速适中，亲切活泼
```

**可选语气风格：**
- `儿童科普风格，语速适中，亲切活泼` — 适合 5-8 岁
- `少儿百科风格，语速正常，清晰自然` — 适合 8-12 岁
- `趣味互动风格，语速稍快，活力充沛` — 适合游戏化场景
- `温柔讲述风格，语速偏慢，耐心引导` — 适合低龄绘本
- 或自定义描述

## 五、各题型特殊字段

### 5.1 拖拽题

```
- 正确答案：B → A → C（用箭头标明正确的拖放顺序）
```

拖拽题自动生成以下额外元素（无需在 PRD 中列出）：
- `final1/2/3...` — 目标放置区域
- `collide1/2/3...` — 碰撞检测区域

### 5.2 连线题

```
- 左侧项：
  - L1：蜂王 | 实拍图+文字 | 蜂王的特写照片
  - L2：工蜂 | 实拍图+文字 | 工蜂采蜜的照片
  - L3：雄蜂 | 实拍图+文字 | 雄蜂的特写照片
- 右侧项：
  - R1：产卵繁殖
  - R2：采蜜筑巢
  - R3：交配后死亡
- 正确连线：L1-R1, L2-R2, L3-R3
```

### 5.3 点选题（不规则热区）

```
- 背景图：实拍图 | 一群企鹅站在冰面上的全景照片
- 热区：
  - 区域1：帝企鹅（画面左侧最大的企鹅）
  - 区域2：帝企鹅（画面右侧抱蛋的企鹅）
- 正确答案：区域1、区域2
```

## 六、省略规则

以下情况可以省略对应字段，AI 会按默认值处理：

| 省略的字段 | 默认行为 |
|-----------|---------|
| 题干图 | 无题干图 |
| 图片描述 | AI 根据选项文字自动生成绘图提示词 |
| 所有动效 | 不生成任何动效 |
| 选项配音 | 文字选项不配音；图+文字选项按文字生成配音 |
| 题干配音 | 按题干文字生成，进入时自动播放 |
| 正确解析配音 | 按解析文字生成，答对后播放 |
| 错误提示配音 | 不生成 |
| 语气要求 | 儿童科普风格，语速适中，亲切活泼 |
| 考查点 | 不记录 |

**最简写法示例（仅填必填项）：**

```
### E1-1 单选题
- 正确答案：B
- 题干：哪个是地球上最大的动物？
- 选项：
  - A：非洲象 | 实拍图+文字
  - B：蓝鲸 | 实拍图+文字
  - C：长颈鹿 | 实拍图+文字
- 正确解析：蓝鲸是地球上最大的动物，体长可达30米。
```
